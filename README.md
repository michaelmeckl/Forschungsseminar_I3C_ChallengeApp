# Forschungsseminar_CoronaSimulation
Projekt des Forschungsseminars "Informationsvisualisierung" im Sommersemester 2020

In der Smartphone-Anwendung sollen aktuelle Informationen, News und Verhaltensweisen zur Coronavirus-Erkrankung (COVID-19) dargestellt werden. Mithilfe von Gamification-Elementen wird der Nutzer fürs Informieren und für richtiges Verhalten belohnt. Um Langzeitmotivation zu schaffen und dem Nutzer zusätzlichen Anreiz zu schaffen, soll er die Möglichkeit haben gegen eine (Corona)-Simulation (basierend auf dem [SIRD-Modell](https://sciencebridge.net/uploads/attatchments/CORONA.SB_.pdf)) anzukämpfen.
