# Forschungsseminar_I3C - ChallengeApp
Projekt des Forschungsseminars "Informationsvisualisierung" im Sommersemester 2020

In der Smartphone-Anwendung werden jeden Tag verschiedene Challenges für den Nutzer angeboten, für deren erfolgreiches Absolvieren er Erfahrungspunkte erhält. Mithilfe von Gamification-Elementen (wie Achievements, Levelsystem, ...) soll der Nutzer dadurch für mehr soziale Interaktion von zuhause aus belohnt werden. Außerdem kann die App durch spannende Challenges mit Freunden immer wieder für Abwechslung und Motivation sorgen.
