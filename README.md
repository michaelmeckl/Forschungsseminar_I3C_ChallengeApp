# Forschungsseminar_I3C - ChallengeApp
Projekt des Forschungsseminars "Informationsvisualisierung" im Sommersemester 2020

In der Smartphone-Anwendung werden jeden Tag verschiedene Challenges für den Nutzer angeboten, für deren erfolgreiches Absolvieren er Punkte und andere Belohnungen erhält, mit denen er beispielsweise seinen In-game Charakter verbessern und neue Möglickeiten freischalten kann. Mithilfe von Gamification-Elementen soll der Nutzer dadurch für soziale Interaktion von zuhause aus belohnt werden. Außerdem kann die App durch spannende Challenges mit Freunden immer wieder für Abwechslung und Motivation sorgen.
